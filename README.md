Alias
=====

[![Build Status](https://travis-ci.org/adjivas/alias.svg)](https://travis-ci.org/adjivas/alias)

This librairy is a systen's project for replace word by alias from environement.

#### Example:
```shell
$ cargo run
with
Ok(", ")
hello
Err("hello")
...
```

```shell

    \_ lib.rs
```
